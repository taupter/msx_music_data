MuSICA用オンラインツール

  • ドラッグ&ドロップでVCDファイルを変換するツール

ドキュメント類

  • FM-BIOS MUSIC DATAフォーマットメモ
  • FMPAC 色々メモ
  • MGSDRVの気になる事メモ
  • MuSICA VCD File Format 解析情報
  • MuSICA ドライバー 音楽データ 仕様書
  • MuSICA ドライバー プログラム 仕様書
  • RTYPEのBGMデータ memo
  • TENPO毎の各音長tick数リスト
  • PSG ハードウェアエンベロープ テクニック メモ
  • PSG トーンレジスタへの設定値
  • TENPO毎の各音長tick数リスト
プロポーショナルフォントに源暎ラテゴ「源暎ラテゴ 詰 v2」を使用しています
等幅フォントに白源フォント「HackGen Console NF」を使用しています

github repository : https://github.com/uniskie/msx_music_data
